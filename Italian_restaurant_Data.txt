 Data 

Based on definition of our problem, factors that will influence decision are:

Number of existing restaurants in the neighborhood (any type of restaurant and especially italian restaurants)
Neighborhood names
Neighborhood Latitude
Neighborhood Longitude
Venue Latitude
Venue Longitude
Venue Category
Population Count
Following data sources will be needed to extract/generate the required information:
Number of restaurants and their type and location in every neighborhood will be obtained using Foursquare API
Coordinate of neighbourhoods will be obtained using Geocoder
Population count of neighborhoods is obtained from WorldAtlas web page
